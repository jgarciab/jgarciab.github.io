Clustering text: comparing TF-IDF, Word2Vec and SBERT embeddings

We are going to cluster the sentences: ["I love cookies", "I love cake", "I hate cookies", "I hate cake", "I like desserts", "My cat's name is Meowzers"]

For that, we need a numeric representation of text, so we create embeddings using TF-IDF, Word2Vec and SBERT:

  • TF-IDF does not work since it's not able to understand that cookies, cake and dessert are similar, and that hate and love are very different
  • Word2Vec does not work very well because the embeddings of love and hate are not too different, so "i love cookies" and "i hate cookies" are more similar than "i love cookies" and "i like desserts"
  • Sentence-BERT (SBERT) works great, because it understands very well that there are sentences liking sweets, sentences hating sweets, and a sentence about a cat

def plot(sentences, embeddings):
    """
    Visualize given embeddings in a 2D space using PCA and annotate them with corresponding sentences.
    
    Parameters:
    - sentences (list of str): A list of sentences corresponding to each embedding.
      The order should match the embeddings provided.
    - embeddings (array-like): A list or array of embeddings, where each embedding should have the same dimension.
    
    Notes:
    - This function makes use of the `adjust_text` library to prevent text labels from overlapping.
    - The plot is displayed using `pylab` (referred to as `plt` in the function).
    
    Returns:
    - A plot with embeddings visualized in a 2D space, annotated with their corresponding sentences.
    
    """
    
    # Reduce the embeddings to 2 dimensions using PCA
    X_embedded = PCA(n_components=2).fit_transform(embeddings)
    
    texts = []
    # Plot each embedding point and its corresponding sentence label
    for text, x, y in zip(sentences, X_embedded[:, 0], X_embedded[:, 1]):
        texts.append(plt.text(x, y, text, fontsize=9))
    plt.plot(X_embedded[:, 0], X_embedded[:, 1], "o")
    
    # Hide the axis
    plt.axis(False)

    # Set the plot limits
    plt.xlim(X_embedded[:, 0].min()*1.8, X_embedded[:, 0].max()*1.8)
    plt.ylim(X_embedded[:, 1].min()*1.8, X_embedded[:, 1].max()*1.8)
    
    # Adjust the position of text labels to avoid overlapping
    adjust_text(texts, arrowprops=dict(arrowstyle='->', color='lightgray'))
